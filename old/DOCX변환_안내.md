# DOCX 변환 안내

## 1. Python 스크립트 실행 (권장)

### Python 설치
1. Python 3.7 이상 설치 필요
2. https://www.python.org/downloads/ 에서 다운로드

### 스크립트 실행
```bash
cd C:\fowardmax
python process_markdown_final.py
```

이 스크립트는 다음 작업을 자동으로 수행합니다:
- 모든 소제목(##, ###, ####)을 볼드 처리
- 모든 `> 참조:` 형식을 각주로 변환
- 모든 `> 주석:` 형식을 각주로 변환
- 표 내의 "소스(근거 문서)" 필드를 각주로 변환
- 목차 항목을 볼드 처리
- 각주 정의를 문서 끝에 추가

## 2. Pandoc 설치 및 DOCX 변환

### Pandoc 설치
1. https://pandoc.org/installing.html 에서 다운로드
2. Windows 설치 프로그램 실행

### DOCX 변환 명령어
```bash
cd C:\fowardmax
pandoc PRD_Phase1_2025-12-31_copy2_processed.md -o "PRD_Phase1_2025-12-31 copy 2.docx" --toc --toc-depth=3 --highlight-style=tango
```

### Pandoc 옵션 설명
- `--toc`: 목차 생성
- `--toc-depth=3`: 목차 깊이 (3단계까지)
- `--highlight-style=tango`: 코드 하이라이트 스타일

## 3. 수동 처리 (Python이 없는 경우)

현재 파일에서 다음 작업을 수동으로 수행해야 합니다:

1. **소제목 볼드 처리**: 모든 `## 제목` 형식을 `## **제목**` 형식으로 변경
2. **참조 각주 변환**: 모든 `> 참조: ...` 형식을 `[^번호]` 형식으로 변경
3. **주석 각주 변환**: 모든 `> 주석: ...` 형식을 `[^번호]` 형식으로 변경
4. **표 내 소스 필드 각주 변환**: 표 내의 "소스(근거 문서)" 필드 값에 `[^번호]` 추가
5. **각주 정의 추가**: 문서 끝에 각주 정의 섹션 추가

## 4. 현재 진행 상황

✅ 완료된 작업:
- 목차 항목 볼드 처리
- 일부 소제목 볼드 처리
- 일부 참조/주석 각주 변환
- 각주 정의 섹션 추가

⏳ 남은 작업:
- 나머지 소제목 볼드 처리
- 나머지 참조/주석 각주 변환
- 표 내 "소스(근거 문서)" 필드 각주 변환

## 5. 참고사항

- 각주 번호는 중복 방지를 위해 자동으로 관리됩니다
- 동일한 내용의 참조/주석은 같은 각주 번호를 사용합니다
- Pandoc 변환 시 각주는 Word의 각주 형식으로 자동 변환됩니다
