# 필드 매핑 매트릭스

**문서명**: output.json 필드 ↔ PRD 필드 매핑 매트릭스  
**버전**: 1.0  
**작성일**: 2025-12-31

---

## 필드 매핑 규칙

1. **의미 기반 매칭**: 필드의 의미와 용도 비교
2. **데이터 타입 일치성**: 필드의 데이터 타입 비교
3. **필수 여부 일치성**: 필드의 필수 여부 비교

---

## IA 필드 ↔ PRD 필드 매핑

| IA 필드명 | IA 필드 타입 | IA 필드 설명 | PRD 필드명 | PRD 필드 타입 | PRD 필드 설명 | 매핑 상태 | 매핑 근거 |
|---|---|---|---|---|---|---|---|
| 기능 ID (System.Xml.XmlElement) | String | Seller-ia-front-{카테고리}-{순번} | Function ID | String | FUNC-## | 매핑 가능 | 기능 식별자 |
| 화면 ID | String | 자유 형식 (카멜케이스) | Screen ID | String | SCR-#### | 매핑 가능 | 화면 식별자 |
| 기능 정의 | String | 기능이 무엇을 하는지 설명 | 기능명/정의 | String | 기능의 한국어 이름 및 정의 | 매핑 가능 | 기능 설명 |
| 카테고리 | String | 기능 카테고리 | Flow ID | String | FLOW-## | 매핑 가능 | 플로우 분류 |
| Depth 1 | String | 최상위 카테고리 | Flow ID | String | FLOW-## | 매핑 가능 | 플로우 분류 |
| Depth 2 | String | 중간 카테고리 | Screen ID | String | SCR-#### | 매핑 가능 | 화면 분류 |
| Depth 3 | String | 세부 카테고리 | Function ID | String | FUNC-## | 매핑 가능 | 기능 분류 |
| 관련 정책 ID | String | dealer-sp-* 형식 | Decision ID / NFR ID | String | D-* / NFR-#### | 매핑 가능 | 정책/요구사항 참조 |
| 중요도 | String | 상/중/하 | - | - | - | 매핑 불가 | PRD에 중요도 필드 없음 |
| 구현 대상 | String/Array | Mobile App / Mobile Web | - | - | - | 매핑 불가 | PRD에 구현 대상 필드 없음 |
| 작업 요소 | String | 디자인/퍼블리싱/개발 | - | - | - | 매핑 불가 | PRD에 작업 요소 필드 없음 |
| 관리자 연동 | String | 관리자 연동 여부 | - | - | - | 매핑 불가 | PRD에 관리자 연동 필드 없음 |

## SPM 필드 ↔ PRD 필드 매핑

| SPM 필드명 | SPM 필드 타입 | SPM 필드 설명 | PRD 필드명 | PRD 필드 타입 | PRD 필드 설명 | 매핑 상태 | 매핑 근거 |
|---|---|---|---|---|---|---|---|
| 정책 코드 | String | dealer-sp-{카테고리}-{순번} | Decision ID / NFR ID | String | D-* / NFR-#### | 매핑 가능 | 정책/요구사항 식별자 |
| 정책 소개 | String | 정책의 간단한 소개 | Decision 내용 | String | 결정 내용 | 매핑 가능 | 정책 설명 |
| 정책 내용 | String | 정책의 상세 내용 | Decision 내용 / NFR 정의 | String | 결정 내용 / 요구사항 정의 | 매핑 가능 | 정책 상세 |
| 정책분류 | String | User/Member, Sign Up, Auth 등 | Decision 카테고리 | String | 결정 카테고리 | 매핑 가능 | 정책 분류 |
| 세부 항목 | String | 정책의 세부 항목 | Decision 세부사항 | String | 결정 세부사항 | 매핑 가능 | 정책 세부 |

## 필드 불일치 항목

### Critical 불일치

| 필드명 | 불일치 내용 | 영향 | 완화 전략 |
|---|---|---|---|
| 기능 ID 형식 | IA: Seller-ia-front-*, PRD: FUNC-## | 자동 매핑 불가 | 매핑 테이블 작성 |
| 화면 ID 형식 | IA: 자유 형식, PRD: SCR-#### | 자동 매핑 불가 | 매핑 테이블 작성 |
| 정책 ID 형식 | SPM: dealer-sp-*, PRD: D-* / NFR-#### | 자동 매핑 불가 | 매핑 테이블 작성 |

### High 불일치

| 필드명 | 불일치 내용 | 영향 | 완화 전략 |
|---|---|---|---|
| 관련 정책 ID | IA의 관련 정책 ID 필드가 대부분 비어있음 | 기능-정책 추적 불가 | 관련 정책 ID 필드 채우기 |
| 정책 내용 | SPM의 정책 내용 필드가 대부분 비어있음 | 정책 상세 파악 불가 | 정책 내용 필드 채우기 |

### Medium 불일치

| 필드명 | 불일치 내용 | 영향 | 완화 전략 |
|---|---|---|---|
| 중요도 | IA에만 존재, PRD에 없음 | 우선순위 관리 불일치 | PRD에 중요도 필드 추가 또는 제거 |
| 구현 대상 | IA에만 존재, PRD에 없음 | 구현 범위 관리 불일치 | PRD에 구현 대상 필드 추가 또는 제거 |
| 작업 요소 | IA에만 존재, PRD에 없음 | 작업 관리 불일치 | PRD에 작업 요소 필드 추가 또는 제거 |
